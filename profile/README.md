# App Builders Gang

We build native Apple platform apps. Our work lives in public repos under this
org.

## Projects

- **[PrayerTimes](https://github.com/App-Builders-Gang/PrayerTimes)** — prayer
  times in your Mac's menu bar. Available on the Mac App Store and via
  Homebrew.

## Infrastructure

- [Shared Homebrew tap](https://github.com/App-Builders-Gang/homebrew-tap) — install any of our apps with `brew install --cask App-Builders-Gang/tap/<app>`
